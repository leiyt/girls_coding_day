# 初识git

### git命令
1. git innit                    初始化一个git仓库
2. git add <file>               add一个文件到仓库
3. git commit -m "des"          提交 -m 后面描述提交改变
4. git status                   查看仓库当前状态
5. git diff                     查看更改
6. git pull URL                 拉取一个请求 URL为实际项目地址
7. git push -u origin master    把更改内容推到主分支
